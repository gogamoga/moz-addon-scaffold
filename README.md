moz-addon-scaffold
==================

Scaffold for Mozilla Addons (sdk, coffee, gulp, mocha, jade, stylus)

Usage
-----
	git clone https://github.com/gogamoga/moz-project-scaffold.git addon_name
	cd addon_name
	npm install
	node init
